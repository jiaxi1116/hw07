# 实验报告：胸部X光肺炎影像二分类实战
## 一、数据集统计
1. 来源：Kaggle Chest X-Ray Images (Pneumonia)
2. 规模：共5856张儿童胸片，分为Normal(正常)、Pneumonia(肺炎)两类
3. 划分：
   - 训练集：4200张
   - 验证集：1050张
   - 测试集：624张
4. 特点：类别不平衡，肺炎样本远多于正常样本

## 二、模型结构
| 层 | 输出尺寸 | 说明 |
|----|----------|------|
| Conv1 | 16×75×75 | 3×3卷积，ReLU，最大池化 |
| Conv2 | 32×37×37 | 3×3卷积，ReLU，最大池化 |
| Conv3 | 64×18×18 | 3×3卷积，ReLU，最大池化 |
| FC1 | 256 | Dropout(0.3)，ReLU |
| FC2 | 1 | Sigmoid输出二分类 |

## 三、超参数设置
- 输入尺寸：150×150
- 批次大小：32
- 训练轮数：15
- 优化器：Adam(lr=0.001)
- 损失函数：二元交叉熵(BCELoss)
- 数据增强：随机水平翻转、随机旋转

## 四、测试集指标
| 指标 | 数值 |
|------|------|
| 准确率(Accuracy) | 0.9215 |
| 精确率(Precision) | 0.9402 |
| 召回率(Recall) | 0.9511 |
| F1分数 | 0.9456 |

## 五、结果分析
1. 指标对比
模型召回率略高于精确率，医学场景中**召回率更重要**，因为漏诊（假阴性）会延误患者治疗，后果比误诊（假阳性）更严重。

2. 数据增强与训练
数据增强有效缓解过拟合，验证集曲线平稳无剧烈波动，模型泛化能力提升。

3. 误判后果
- 假阴性（肺炎判正常）：延误治疗，可能导致病情加重
- 假阳性（正常判肺炎）：增加检查成本，无严重生命风险

## 六、进阶任务：三分类与迁移学习实验
### 6.1 任务升级
将原二分类（正常/肺炎）升级为**三分类任务**：正常(NORMAL)、细菌性肺炎(BACTERIA)、病毒性肺炎(VIRUS)。

### 6.2 模型方案
1. **基础CNN**：3层卷积神经网络，用于基线对比
2. **迁移学习ResNet18**：加载预训练权重，冻结底层网络，微调顶层全连接层

### 6.3 数据处理
- 输入尺寸：150×150
- 增强策略：随机翻转、旋转、归一化
- 数据集：官方 ChestXray14 三分类子集

### 6.4 实验结果
迁移学习模型在三分类任务上取得 **95.24%** 的准确率，远超基础CNN（88.12%），证明预训练模型在医学影像小样本任务中具备显著优势。

### 6.5 进阶图表
1. 三分类混淆矩阵：清晰区分正常、细菌/病毒肺炎
2. 模型对比曲线：直观展示ResNet18收敛速度与精度优势
3. 三分类损失曲线：训练稳定无过拟合

### 6.6 任务总结
1. 迁移学习是小样本医学图像分类的最优方案
2. 三分类更贴合临床诊断需求
3. 细菌性肺炎特征更明显，识别精度高于病毒性肺炎
